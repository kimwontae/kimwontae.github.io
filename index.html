4차 산업혁명과 IoT

4차 산업혁명

4차 산업혁명이란 인공지능, 사물 인터넷(IOT), 빅데이터, 모바일 등 첨단 정보통신기술이 경제, 사회 전반에 융합되어 혁신적인 변화가 나타나는 차새대 산업혁명이다.

IoT

IoT란 세상의 모든 사물들이 네트워크로 '연결'되어 실시간으로 데이터를 주고받는기술이다. 이는 집안의 사물들을 PC, 핸드폰등으로 인터넷이 연결된곳에서 제어를 가능하게 한다.

구동원리

작품의 구동원리

  • 기본구동

    NODE MCU에 내장된 ESP8266칩을 사용하여서 WIFI에 접속 후 MCU고유의 IP주소로 html의 언어를 텍스트 형식으로 보내주어 웹 브라우저를 구성.

  • 기본구동

    웹 브라우저에 버튼 누를 때의 텍스트의 인덱스 값을 읽어서 각각의 맞는 수행을 하게 됨.

  • 수동모드, 자동모드 선택

    자동모드시

    CDS(조도센서)값에 따라 밝기가 밝으면 블라인드를 올리고 LED를 끄고, 어둡다면 블라인드를 내리고 LED를 점등함

  • 수동모드, 자동모드 선택

    수동모드시

    스마트폰이나 PC의 웹 페이지에서 LED와 Servo motor, Stepper motor를 동작시켜 각각 현관문, 블라인드를 수동으로 제어할수있다.

Node MCU(ESP8266)

WIFI모듈

NODE MCU(ESP8266)은 WIFI모듈로써 소자 고유의 IP값을 지정해주고 이 IP주소를 이용하여 기본적인 웹 페이지를 만들수 있다. 기존에 사용하던 Arduino IDE를 이용하여 개발 가능하고 또한 웹 페이지는 HTML를 통하여 개발가능하다.

Stepper motor

블라인드 제어

Stepper motor는 작품의 블라인드를 제어하는데 사용하였다. Stepper motor는 DC motor와 다르게 한 바퀴의 회전을 많은 수의 스텝들로 나눌수있어 아주 정확하게 조절이 가능하다. 블라인드를 오르고 내릴때 정확하게 오르고 내리지 못하면 블라인드에 고장이 생길수있는데 DC motor를 사용하면 정확하게 조절할수없기 때문에 Stepper motor를 사용했다.

Servo motor

현관문 제어

Servo motor는 작품의 현관문을 제어하는데 사용하였다. 기존에 사용하던 0~180도 까지 제어가능한 기본 Servo motor를 사용하였고 각도를 제어하여 문을 열었다 닫는 동작을 구현하였다.

CDS 조도센서

빛의양 측정

자동모드 일때 CDS 조도센서를 이용하여 실시간으로 일조량을 넘겨받아 일정량이상(즉 낮이나 햇빛이 강함)이면 블라인드를 내리도록 설정하였다.

L9110s

Stepper motor를 제어하기 위한 모터 드라이버로 일전에 라인트레이서구동 당시 사용했던 모터드라이버이다.
